How It Works

This visitor counter is implemented using PHP to track the number of visitors to the website. Here's how it works:

  1. PHP creates or reads a file that stores the visitor count.
  2. Each time the page loads, the count is incremented by one.
  3. The updated count is saved back to the file for future visits.
  4. The current count is displayed on the page.

Note

This is a demonstration of how a PHP visitor counter would work. In a real implementation, this would be powered by a PHP script on the server. The current count shown is a placeholder since this is a static HTML page in your portfolio.

The actual PHP code would look like this:

<?php
// File to store the count
$counterFile = "counter.txt";

// Check if the file exists
if (!file_exists($counterFile)) {
    // If not, create it with a count of 1
    $handle = fopen($counterFile, "w");
    fwrite($handle, "1");
    fclose($handle);
    $count = 1;
} else {
    // If it exists, read the current count
    $handle = fopen($counterFile, "r");
    $count = fread($handle, 20);
    fclose($handle);
    
    // Increment the count
    $count++;
    
    // Write the new count back to the file
    $handle = fopen($counterFile, "w");
    fwrite($handle, $count);
    fclose($handle);
}

// Display the count
echo "<h3>Total Visitors</h3>";
echo "<div class='counter-value'>$count</div>";
echo "<p>You are visitor number $count to this website!</p>";
?>
