Note

This is a demonstration of how a PHP digital clock would work. In a real implementation, this would be powered by a PHP script on the server. The current time shown is a placeholder since this is a static HTML page in your portfolio.

The actual PHP code would look like this:

<?php
// Set the timezone
date_default_timezone_set('Asia/Kolkata');

// Get the current date and time
$currentDate = date('l, F j, Y');
$currentTime = date('h:i:s A');

// Display the date and time
echo "<div class='date-display'>$currentDate</div>";
echo "<div class='clock-display'>$currentTime</div>";

// Server information
echo "<div class='timezone-info'>";
echo "<h3>Server Information</h3>";
echo "<p><strong>Timezone:</strong> " . date_default_timezone_get() . " (IST)</p>";
echo "<p><strong>Server:</strong> " . $_SERVER['SERVER_SOFTWARE'] . "</p>";
echo "<p><strong>PHP Version:</strong> " . phpversion() . "</p>";
echo "</div>";
?>
